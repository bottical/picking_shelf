1. はじめに

本ページでは、シェルフローの基本操作と、作業中によく発生する確認事項を説明します。

基本的な流れ

  1. ログインします。
  2. CSVデータを取り込みます。
  3. 作業画面でバーコードを読み取ります。
  4. 数量や完了状態を確認します。
  5. 必要に応じて進捗や完了データを確認します。

推奨環境

  • 最新版のGoogle ChromeまたはMicrosoft Edgeを推奨します。
  • バーコードリーダーを使用する場合は、読み取り先の入力欄にカーソルがあることを確認してください。
  • 通信が不安定な場合、CSV/Excel取込や完了状態の反映に時間がかかることがあります。

2. ログイン方法

  1. ログイン画面を開きます。
  2. メールアドレスとパスワードを入力します。
  3. ログインボタンを押します。

ログインできない場合

  • メールアドレス・パスワードに誤りがないか確認してください。
  • ブラウザを再読み込みしてください。
  • 解決しない場合は管理者へ連絡してください。

3. CSV取込

  1. CSV取込画面を開きます。
  2. 対象のCSVファイルを選択します。
  3. マッピング設定や表示内容を確認します。
  4. 取込を実行します。

CSVが読み込めない場合

  • ファイル形式がCSVまたは指定形式になっているか確認してください。
  • 必要な列が不足していないか確認してください。
  • Excelで保存したファイルの場合、文字コードや改行形式の影響で読み込めない場合があります。

4. 検品作業

  1. 検品・投入画面を開きます。
  2. ピッキングIDを入力または読み取ります。
  3. 対象商品が表示されたことを確認します。
  4. 商品バーコードを読み取ります。
  5. 必要数量に達すると、その商品が完了状態になります。

5. 仕分け作業

  1. 卸仕分けモードを開きます。
  2. 必要に応じて作業者や対象データを選択します。
  3. 対象SKUのバーコードを読み取ります。
  4. 表示された卸先ごとの数量を確認します。
  5. 投入が完了した卸先の完了ボタンを押します。
  6. 誤って完了した場合は、完了表示の長押しで取消を行います。

表示数量の見方

  • 卸先ごとの数量を確認し、表示された数量分を投入してください。
  • 0数量の行は、その卸先に投入する対象がないことを示します。
  • 完了後に誤りに気付いた場合は、対象カードの「長押しで取消」表示に従って完了を取り消してください。

6. 進捗確認

進捗画面では、作業の完了状況や未完了状況を確認できます。

  • 完了済みの作業
  • 未完了の作業
  • 作業者別の状況
  • 必要に応じたCSV出力

7. よくある質問

バーコードを読み取っても反応しません

  • 入力欄にカーソルが入っているか確認してください。
  • バーコードリーダーが接続されているか確認してください。
  • 別のバーコードで反応するか確認してください。

「Excel取込ライブラリを読み込めませんでした」または「Excel読込ライブラリの読み込みに失敗しました」と表示されます

  • 通信環境が不安定な場合は、画面を再読み込みしてから再度取り込んでください。
  • 急ぎの場合は、ExcelファイルをCSV形式で保存し直して取り込んでください。
  • 時間をおいても解消しない場合は、ブラウザのキャッシュ削除または別端末で確認してください。
  • 同じ現象が続く場合は、発生日時・使用端末・画面キャプチャを管理者へ連絡してください。

画面が更新されません

  • ブラウザを再読み込みしてください。
  • 通信環境を確認してください。
  • 同じ現象が続く場合は管理者へ連絡してください。

完了一覧に表示されません

  • 作業が完了状態になっているか確認してください。
  • 検索条件や日付条件を確認してください。
  • 画面を再読み込みしてください。
